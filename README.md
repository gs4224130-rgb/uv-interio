# UV INTERIO — Direct Upload V3

Premium responsive static website for UV INTERIO, Tijara.

## Easy image replacement
This package uses flat image filenames so you do not need to create `assets/images` folders.

Replace any image by uploading a new file with the same filename:
- `panel-01.webp` ... `panel-20.webp`
- `light-01.webp` ... `light-10.webp`
- `showroom-front.webp`
- `showroom-wide.webp`
- `showroom-interior.webp`
- `owner-gourav.webp`

See `IMAGE-REPLACEMENT-GUIDE.txt`.

## GitHub upload
Upload all files in this folder directly to the root of the `uv-interio` repository. Keep `CNAME`.
