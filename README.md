# Interior Showroom Website

## File structure

interior-showroom-site/
├── index.html
├── about.html
├── services.html
├── contact.html
├── styles.css
├── main.js
└── README.md

## Customize before launch

1. Replace `[Insert Showroom Name]` everywhere with your final brand/showroom name.
2. Replace phone, email, PIN code and opening hours placeholders.
3. Replace the owner placeholder portrait on `about.html` with a real photo.
4. Replace Unsplash image URLs with your own showroom/product photos for stronger trust.
5. Add the exact Google Maps link/embed on `contact.html`.
6. Connect the contact form to Formspree, Netlify Forms, your server, or another form backend.
7. Update colors and typography from the CSS variables at the very top of `styles.css`.

## Preview locally

Open `index.html` directly in a browser, or run a simple local server.
