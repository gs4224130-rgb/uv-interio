# UV INTERIO — Website V2

Premium responsive static website for UV INTERIO, Tijara.

## Main files
- `index.html` — homepage
- `about.html` — brand / owner page
- `services.html` — collections page with 20 panel references + 10 lighting references
- `contact.html` — showroom/contact page
- `styles.css` — complete design system and responsive styling
- `catalog-data.js` — easy-to-edit panel and lighting catalogue data
- `main.js` — navigation, reveal effects, filters, parallax and interactions
- `CNAME` — preserves `www.uvinterio.com` on GitHub Pages

## Branding
All main colours, typography and layout tokens are CSS variables at the top of `styles.css`.

## Photos
The showroom and owner photos are the user-provided UV INTERIO images. The panel and lighting sample cards are cropped from the showroom photos to keep the site authentic and avoid using random copyrighted Pinterest/Google imagery.

## Before launch
Replace these placeholders in `contact.html` with confirmed details:
- phone number
- WhatsApp number
- opening hours

The enquiry form is front-end only. Connect it to Formspree, Netlify Forms or another backend before expecting submissions.

## GitHub Pages update
Upload/replace all files in the repository root. Keep the `CNAME` file so the custom domain remains `www.uvinterio.com`.
